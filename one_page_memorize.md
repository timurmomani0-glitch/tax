# One-Page Memorization Sheet (Exam-Execution Version)

## Universal Answer Order
1) Classify item
2) State rule
3) Apply numbers
4) Show working
5) Conclude liability

## Must-Memorize Engines
- Income tax: Total -> Net -> Taxable -> Band split (NSI/SI/DI) -> Credits
- Trading: Accounts profit + disallowables - allowables -> adjusted profit -> CA
- CGT: Proceeds - costs - reliefs -> net gains/losses -> AEA -> rates
- CT: Trading + Property + Gains - reliefs = TTP -> rate framework
- VAT: Output - recoverable input

## High-Risk Traps
- Wrong income-category allocation
- Missing PA taper logic
- Forgetting depreciation add-back
- Wrong disposal value in CA pool
- Zero-rated vs exempt confusion
- Wrong NIC class/threshold basis
- Missing overlap relief step

## Standard Examiner Wording
- "Add back as non-deductible (not wholly and exclusively for trade)."
- "Deduct as allowable revenue expense."
- "Allocate remaining basic-rate band before higher rate."
- "Input VAT blocked due to exempt-use restriction."

## Timed Checklist (per question)
- Proforma drawn?
- Every figure labelled with rule?
- Rates from tax table checked?
- Final answer rounded/presented clearly?
